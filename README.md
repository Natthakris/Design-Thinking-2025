# [ดูแลตนเองให้ถูกวิธี - Self Caring]

แอปพลิเคชั่นเกี่ยวกับบทความเกี่ยวกับสุขภาพและการป้องกันโรคเพื่อให้ความรู้ นิสิต นักศึกษา และบุคคลากรในมหาวิทยาลัยพะเยา และช่วยแก้ปัญหาด้านสุขภาพ ความเครียด และพฤติกรรมสุขภาพ

## ฟีเจอร์หลัก (Features)
- ฟีเจอร์ที่ 1 การออกแบบที่ใช้งานง่าย
- ฟีเจอร์ที่ 2 การให้ความรู้ด้านสุขภาพ 

## สมาชิกและภาระงาน (Team Members & Responsibilities)

ตารางแสดงการแบ่งงานรายสัปดาห์:

| สัปดาห์ที่ (Week) | ณัฐกฤษ |
| :--- | :--- |
| **Week 1** | วางแผนโครงงาน และรวบรวมความต้องการ  ออกแบบ UI/UX บน Streamlit (Mockup) |
| **Week 2** | จัดเตรียมและทำความสะอาดข้อมูล (Data Preprocessing)  พัฒนาส่วนเชื่อมต่อกับผู้ใช้ (Frontend Layout) |
| **Week 3** | พัฒนาโมเดลหรือลอจิกเบื้องหลัง (Backend Logic)  เชื่อมต่อโมเดลเข้ากับหน้าเว็บ Streamlit |
| **Week 4** | ทดสอบระบบ (Testing) และแก้ไข Bug  จัดทำเอกสารประกอบโครงงาน และ Deploy |
